## 예제 실행  

### GUI 환경에서 실행  

Thonny 편집기로 .py 파일을 열고 실행(Run)버튼으로 프로그램을 실행해 줍니다.  

### CLI (터미널)환경에서 실행  

터미널 창에서 파일이 있는 디렉토리로 이동, 아래의 명령어로 실행해 줍니다.  

```bash
sudo python3 switch.py  
```
